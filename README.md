# AED
C language 
Final program AED - (Supermercado)
O objetivo deste trabalho é desenvolver um programa para efetuar a gestão de um dia de atividade dum grande supermercado:
  -O supermercado tem um diversificado stock de produtos, e há reabastecimentos ocasionais.
  -Os clientes circulam pelos corredores escolhendo e colocando produtos nos seus cestos de compras.
  -No final das suas compras, cada cliente vai para uma caixa efetuar o pagamento.
  -O supermercado mantém um histórico com estatísticas sobre as vendas individuais.

Comando:
? - mostra os comandos disponíveis.
C - Cliente novo entra no supermercado.
D x p n - O cliente x acrescenta n unidades do produto p ao seu cesto.
E - Lista todos os clientes que estão no supermercado por ordem de chegada, escrevendo para cada cliente: a designação, se está numa caixa e qual, o conteúdo do cesto por ordem alfabética. Detalhes nos exemplos de execução.
P p n c g - Novo produto acrescentado no stock, com designação p, quantidade n, preço c e peso g.
Q p n - O stock do produto existente p é incrementado n unidades.
R - Lista o stock por ordem alfabética, escrevendo toda a informação dos produtos em stock. Detalhes nos exemplos de execução.
. - Termina imediatamente a execução do programa, sem se preocupar com as caixas e os clientes.
X x n - O cliente x junta-se à caixa n.
Y x - O cliente x junta-se à caixa com menor tempo de espera.
Z - Lista todas as caixas por ordem, indicando para cada caixa o seu índice e os clientes que estão na fila. Detalhes nos exemplos de execução.
H - lista o Histórico corrente pela ordem pela qual foram acrescentados os registos. Detalhes nos exemplos de execução.
T s - avança o Tempo s segundos.
F - Fecha o supermercado e finaliza a execução do programa.
